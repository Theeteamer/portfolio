# Ian Byegon Portfolio

A responsive portfolio website built with HTML, CSS, and JavaScript.

## Project structure

- `index.html` — main page markup
- `css/style.css` — styling and responsive layout
- `js/main.js` — menu toggle and scroll reveal logic
- `assets/` — profile illustration and resume placeholder

## Notes

- The site includes a sticky navigation menu, section anchors, and animated cards.
- The resume download currently uses a placeholder file in `assets/resume.txt`.
- Replace placeholder assets with real profile image and resume content when ready.
